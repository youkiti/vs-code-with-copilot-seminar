# 2. 準備：必要なものをインストールしよう

このセクションでは、データ分析に必要なソフトをパソコンにインストールします。
初めて見る言葉がたくさん出てきますが、一つずつ順番に進めていけば大丈夫です。

## このセクションでやること

1. **プログラムを書くための道具を用意**
   - VS Code（高機能なメモ帳）をインストール
   - 日本語化と便利な機能の追加

2. **プログラムを保存する場所を作る**
   - GitHub（プログラムのSNS）のアカウントを作成
   - AIアシスタント（Copilot）の設定

3. **プログラムを管理するツールを入れる**
   - Git（作業を記録するカメラのようなもの）をインストール
   - 基本的な設定

## 2.1 VS Code（プログラムを書く場所）のセットアップ

VS Codeは、プログラムを書くための高機能なメモ帳です。
メモ帳と違って、プログラムを書くのに便利な機能がたくさんついています。

### インストールの仕方

1. [VS Code公式サイト](https://code.visualstudio.com/)にアクセス
2. 「Download」ボタンをクリック
   - Windowsの人：自動的にWindows用がダウンロードされます
   - Macの人：Mac用が表示されるのでそれをダウンロード
3. ダウンロードしたファイルをダブルクリック
4. 画面の指示に従ってインストール
   - 基本的に「次へ」を押していけばOKです

### 日本語にする

最初は英語で表示されますが、日本語に変更できます：

1. VS Codeを起動する
2. 左側のパズルのピースみたいなアイコン（Extensions）をクリック
3. 上の検索窓に「Japanese」と入力
4. 「Japanese Language Pack」をインストール
   - 「Install」ボタンをクリック
5. VS Codeを一度終了して、もう一度起動
   - これで日本語になります

### 便利な機能を追加する

VS Codeは、必要な機能を追加できます。
以下の機能を順番に追加していきましょう：

#### R言語を使えるようにする
1. 左側のパズルのピースアイコンをクリック
2. 以下の2つを順番にインストール：
   - 検索窓に「R Extension」→「Install」
   - 検索窓に「R LSP」→「Install」

#### GitとGitHubを使えるようにする
1. 同じくパズルのピースアイコンから：
   - 検索窓に「GitLens」→「Install」
   - 検索窓に「GitHub Pull Requests」→「Install」

インストールが終わったら：
- VS Codeを一度終了
- もう一度起動する
これで新しい機能が使えるようになります！

## 2.2 GitとGitHubについて

プログラミングでは、コードの変更履歴を管理することが重要です。GitとGitHubは、その管理を助けてくれるツールです。

### GitとGitHubの違い

#### Git（ローカル）
- **自分のPC上で動くシステム**
- 例えるなら、「デジタルカメラ」のようなもの
  * 写真（コードの変更）を撮影（記録）できる
  * 撮影した写真（変更履歴）を見返せる
  * 必要なら前の状態に戻せる

#### GitHub（オンライン）
- **インターネット上のサービス**
- 例えるなら、「Instagramのようなサービス」
  * 写真（コード）をオンラインで保存できる
  * 他の人と共有できる
  * 他の人の写真（コード）を見ることもできる

### 使い方の例
1. Gitで自分のPCに変更を記録（撮影）
2. GitHubにアップロード（投稿）
3. 他の人と共有したり、別のPCでも使えるように

## 2.3 GitHubアカウントの作成とCopilotの設定

### GitHubアカウントの作成

最初に、コードを管理・共有するためのGitHubアカウントを作成します。

1. [GitHub](https://github.com/)にアクセス
2. 「Sign up」をクリック
3. 必要事項を入力：
   - Username（ユーザー名）：他の人と重複しない名前
   - Email（メールアドレス）：確認メールが届くアドレス
   - Password（パスワード）：セキュアなパスワード
4. アカウント作成後、メールアドレスの確認を行う
   - 登録したメールアドレスに確認メールが届きます
   - メール内のリンクをクリックして確認完了

### GitHub Copilotの設定

GitHubアカウントを作成したら、すぐにGitHub Copilotも設定しましょう：

1. [GitHub Copilot](https://github.com/features/copilot)にアクセス
2. 「Get started for free」をクリックし、設定
3. VS Codeで拡張機能をインストール：
   - 左側のExtensionsアイコンをクリック
   - 検索バーに「GitHub Copilot」と入力
   - "GitHub Copilot"をインストール
4. VS CodeでGitHubアカウントにサインイン：
   - 画面左下の人型アイコンをクリック
   - 「Sign in with GitHub」を選択
   - ブラウザが開くので、認証を完了

## 2.3 Git（作業を記録するツール）のセットアップ

Gitは、プログラムの変更を記録するためのツールです。
デジタルカメラのように、作業の記録を残すことができます。

### インストールの仕方

#### Windowsの人
1. [Git for Windows](https://gitforwindows.org/)のサイトにアクセス
2. 大きな「Download」ボタンをクリック
3. ダウンロードしたファイルをダブルクリック
4. インストール画面が出てきたら：
   - 基本的に「Next」を押していけばOK
   - 分からない設定は、そのままでOK

#### Macの人
1. 「ターミナル」を開く
   - Finderの「アプリケーション」→「ユーティリティ」→「ターミナル」
2. 以下のコマンドを入力：
   ```bash
   git --version
   ```
3. Gitがない場合は、インストールの案内が表示される
   - 画面の指示に従ってインストール

### 初期設定（GitHubとの連携）

GitとGitHubを連携させて、作業を記録・共有できるようにします：

1. ターミナル（またはGit Bash）を開く方法：
   - Windowsの人：デスクトップの「Git Bash」アイコン
   - Macの人：「ターミナル」アプリ

2. 以下のコマンドを順番に入力：
```bash
# 1. GitHubで登録した名前を設定
git config --global user.name "あなたの名前"

# 2. GitHubで登録したメールアドレスを設定
git config --global user.email "あなたのメールアドレス"
```

### 作業場所（リポジトリ）を作る

設定が終わったら、プログラムを保存する場所を作ります。
GitHubでは、この保存場所のことを「リポジトリ」と呼びます。

1. GitHubのウェブサイトにログインする
2. 画面右上の「+」マークをクリック
3. メニューから「New repository」を選ぶ
4. 必要な情報を入力：
   - Repository name（名前）：
     * 例：「my-first-project」
     * 英語で分かりやすい名前をつける
   - Description（説明）：
     * どんなプロジェクトか簡単に書く
     * 日本語でOK
   - 公開設定を選ぶ：
     * Private（非公開）：自分だけが見られる
     * Public（公開）：誰でも見られる
     * ※あとで変更できます
5. 「Create repository」（作成）ボタンをクリック

これで、プログラムを保存する場所ができました！

## 困ったときは

### よくある問題と解決方法

1. **VS Codeの拡張機能がインストールできないとき**
   - インターネットに接続されているか確認する
   - VS Codeを一度終了して、もう一度起動する
   - それでも解決しない場合は、PCを再起動する

2. **Gitのコマンドが動かないとき**
   - Windowsの場合：
     * スタートメニューから「Git Bash」を探して起動する
     * それでもダメなら、PCを再起動する
   - Macの場合：
     * ターミナルを開き直す
     * それでもダメなら、PCを再起動する

3. **GitHubにログインできないとき**
   - ユーザー名とパスワードを確認する
   - メールアドレスの確認が済んでいるか確認する
   - ブラウザのキャッシュをクリアしてみる

### 解決しないときは

ChatGPTに質問してみましょう。エラーメッセージをそのままコピー＆ペーストして、
「このエラーの解決方法を教えてください」と聞いてみてください。
